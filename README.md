# 🛡️ CekFakta AI - Gateway Edukasi & Deteksi Penipuan Siber

**CekFakta AI** adalah aplikasi berbasis web pintar yang dirancang sebagai gerbang pertolongan pertama (*First-Aid*) bagi masyarakat awam untuk mendeteksi potensi hoaks, penipuan digital (seperti modus bansos, kuota gratis), serta manipulasi informasi yang beredar di media sosial atau grup pesan instan.

Proyek ini dikembangkan secara adaptif dalam rangka kompetisi **#JuaraVibeCoding**.

---

## 🎯 Masalah & Target Audiens

Di Indonesia, penyebaran hoaks dan penipuan digital bermodus bantuan sosial (bansos) atau bagi-bagi kuota internet gratis sangat merugikan masyarakat awam—terutama orang tua dan pengguna internet baru—yang belum memiliki kemampuan literasi digital yang cukup. 

**CekFakta AI hadir untuk menyelesaikan masalah ini dengan berfokus pada:**
* **Target Audiens:** Masyarakat awam dan pengguna umum yang membutuhkan validasi instan tanpa alur yang rumit.
* **Dampak Nyata:** Memberikan perlindungan siber dini agar pengguna tidak terjebak mengklik tautan *phishing* atau membagikan data sensitif mereka.

---

## 🚀 Fitur Utama & Proposisi Nilai (Value Proposition)

Aplikasi ini mengusung pendekatan *Cyber-Minimalist* yang bersih, intuitif, dan responsif dengan fitur-fitur unggulan:

1.  **AI Hoax & Risk Analyzer:** Menganalisis indikasi kebohongan teks menggunakan model AI canggih (`gemini-1.5-flash`) untuk menghasilkan *Trust Score* dan tingkat risiko (*Risk Level*) secara cepat.
2.  **Cyber-Safety Action Plan:** Tidak sekadar mendeteksi, aplikasi langsung merumuskan panduan taktis darurat (*Immediate Action*) dan ciri fisik hoaks (*Hoax Red Flags*) agar pengguna awam tahu apa yang harus dilakukan detik itu juga.
3.  **Kanal Konfirmasi Mandiri (Anti-404):** Menyediakan tombol akses cepat yang dinamis dan aman langsung ke beranda pencarian institusi validasi tepercaya di Indonesia (Komdigi, CekFakta.com, dan Kompas CekFakta) untuk verifikasi lebih lanjut.
4.  **Auto-Clear UX Enhancement:** Antarmuka otomatis membersihkan input setelah analisis selesai untuk menjaga alur pencarian pengguna tetap bersih dan nyaman.

---

## 🛠️ Tech Stack & Arsitektur

Aplikasi ini dibangun menggunakan kombinasi teknologi modern standar industri:
* **Frontend & Backend:** Next.js (App Router) & TypeScript
* **Styling:** Tailwind CSS (Cyber-Minimalist Theme)
* **AI Engine:** Google Gemini API SDK (`gemini-1.5-flash`)
* **Search Engine Integration:** Google Custom Search API (Terintegrasi ke portal penanganan hoaks resmi)
* **Infrastruktur & Deployment:** Google Cloud Run (Continuous Deployment via GitHub)

---

## 📦 Jalur Kepatuhan Hukum (Legal Compliance)

Sebagai aplikasi yang mengedukasi masyarakat mengenai keamanan digital, CekFakta AI dirancang dengan matang hingga ke aspek hukum produk:
* **Kebijakan Privasi (Privacy Policy):** Menjamin bahwa setiap data teks atau gambar yang diinput pengguna tidak direkam secara permanen di database demi privasi mutlak.
* **Syarat Layanan (Terms of Service):** Menegaskan batasan tanggung jawab hasil analisis berbasis AI sebagai panduan edukatif awal.
